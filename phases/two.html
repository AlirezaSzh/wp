گزارش فاز دوم پروژه طراحی افزونه وردپرس

تاریخ شروع فاز دوم: ۱۴۰۳/۰۹/۲۸
تاریخ پایان فاز دوم: ۱۴۰۳/۱۰/۰۷
هدف فاز دوم: پیاده‌سازی قابلیت‌های مربوط به فیلدهای داینامیک، نمایش نقشه، و سیستم جستجو و فیلتر پیشرفته.

جزئیات کارهای انجام‌شده در فاز دوم

1. فیلدهای داینامیک

  1. طراحی سیستم مدیریت فیلدهای داینامیک در پنل مدیریت وردپرس:

    • ایجاد ماژولی در بخش مدیریت وردپرس که مدیر بتواند فیلدهای دلخواه برای هر دسته‌بندی تعریف کند.
    • رابط کاربری ساده و تعاملی برای افزودن، ویرایش، و حذف فیلدها.

  2. اتصال فیلدهای داینامیک به دسته‌بندی‌ها:

    • پیاده‌سازی امکان اتصال فیلدهای سفارشی به دسته‌بندی‌های مختلف ایونت‌ها و کسب‌وکارها.

  3. ذخیره‌سازی اطلاعات فیلدهای داینامیک:

    • طراحی و پیاده‌سازی سیستم ذخیره‌سازی اطلاعات فیلدهای سفارشی در پایگاه داده.
    • امکان مشاهده اطلاعات ذخیره‌شده در پیش‌خوان وردپرس.

  4. نمایش فیلدهای داینامیک در صفحات عمومی سایت:

    • نمایش اطلاعات وارد شده توسط کاربران در صفحات عمومی مرتبط با ایونت‌ها و کسب‌وکارها.

2. نمایش نقشه

  1. ادغام با OpenStreetMap:

    • پیاده‌سازی ادغام با سرویس نقشه (OpenStreetMap).
    • استفاده از API برای نمایش نقشه در صفحات سایت.

  2. نمایش ایونت‌ها و کسب‌وکارها روی نقشه:

    • افزودن قابلیت نمایش ایونت‌ها و کسب‌وکارها روی نقشه با استفاده از آیکن‌های سفارشی.
    • طراحی سیستم دسته‌بندی آیکن‌ها براساس نوع (ایونت یا کسب‌وکار).

  3. جستجو در نقشه:

    • افزودن قابلیت جستجو براساس محدوده یا موقعیت مکانی کاربر.
    • امکان جابجایی روی نقشه و به‌روزرسانی نتایج به‌صورت خودکار.

  4. بهینه‌سازی عملکرد نقشه:

    • بهینه‌سازی بارگذاری نقشه و داده‌ها برای کاهش زمان بارگذاری.
    • مدیریت درخواست‌های API برای کاهش هزینه‌ها و استفاده بهینه از منابع.

3. جستجو و فیلتر پیشرفته

  1. طراحی سیستم جستجو:

    • پیاده‌سازی قابلیت جستجو براساس کلمات کلیدی، دسته‌بندی‌ها، و مکان.
    • استفاده از WP_Query برای بهینه‌سازی جستجو.

  2. ایجاد سیستم فیلتر:

    • طراحی قابلیت فیلتر براساس فیلدهای خاص مانند تاریخ، نوع (ایونت یا کسب‌وکار)، و مکان.
    • رابط کاربری برای فیلترها در صفحات عمومی.

  3. ادغام AJAX:

    • استفاده از AJAX برای بارگذاری نتایج جستجو و فیلتر بدون نیاز به رفرش صفحه.
    • به‌روزرسانی پویا نتایج هنگام تغییر فیلترها یا واردکردن کلمات کلیدی.

  4. بهینه‌سازی پایگاه داده:

    • بهینه‌سازی کوئری‌ها برای افزایش سرعت جستجو و فیلتر.
    • استفاده از ایندکس‌گذاری مناسب روی جداول پایگاه داده.

دستاوردهای فاز دوم

  1. سیستم مدیریت فیلدهای داینامیک:

    • امکان افزودن فیلدهای خاص برای دسته‌بندی‌های مختلف توسط مدیر.
    • نمایش اطلاعات فیلدهای سفارشی در صفحات عمومی.

  2. قابلیت نمایش نقشه:

    • نمایش ایونت‌ها و کسب‌وکارها روی نقشه با جستجوی محدوده‌ای.
    • طراحی آیکن‌های سفارشی برای هر نوع داده.

  3. جستجو و فیلتر پیشرفته:

    • امکان جستجو و فیلتر پیشرفته براساس نیاز کاربران.
    • بارگذاری نتایج به‌صورت پویا با استفاده از AJAX.
